# Changelog

## v0.3 (2026-06-21)

### 📂 拖入文件夹功能

- **源文件夹和目标目录支持拖入**：从资源管理器将文件夹拖到输入框上，自动填入路径
- **新增 `DropLineEdit` 类**：继承 `QLineEdit`，重写拖放事件，只接受文件夹

---

## v0.2 (2026-06-21)

### 🗂️ 扩展映射升级

- **预设映射从 7 条扩充至 31 条**：按视频(7)、音乐(6)、图片(7)、文档(8)、压缩包(3) 分组排列
- **每行增加「启用」勾选框**：用户自由选择生效的映射，`_build_config` 只收集勾选行
- **默认全部不勾选**：首次打开无预设勾选，避免误操作
- **状态持久化**：勾选状态、用户新增的行均通过 QSettings 数组读写保存，关闭窗口或执行后自动恢复

### 🎨 UI 优化

- **列宽 1:2 比例**：扩展名列与文件夹名列按 1:2 分配宽度，并在窗口显示/缩放时动态调整
- **窗口最小宽度从 800 降至 640**：整体 UI 随列宽缩小
- **表格初始高度调整为 3 行**：刚好显示 3 行 + 表头

---

## v1.1.0 (2026-06-21)

### ✨ PySide6 图形界面

新增 `main.py` 图形界面，覆盖全部核心配置，后台线程执行不阻塞 UI。

#### 新增

- **图形界面**：基于 PySide6 的主窗口，表单化配置全部参数
- **文件夹选择**：源文件夹/目标目录均支持浏览选择

- **实时日志**：重定向 stdout 到日志面板，实时显示执行过程

## v1.0.0 (2026-06-21)

### ✨ 初始版本

首个完整可用版本，实现文件分类→重命名→分组压缩→二次打包的完整流程。

#### 功能特性

- **文件分类**：按扩展名自动归类，支持自定义文件夹名映射
- **顺序重命名**：支持按文件名或修改时间排序，自动处理命名冲突
- **分组压缩**：
  - 调用 Bandizip 命令行进行分卷压缩
  - 自动检测分卷大小（<4GB 对半切，≥4GB 每卷 4GB）
  - 支持手动指定分卷大小
  - 支持密码保护
- **二次打包**：将分卷文件再次压缩为 `.zipp` 单文件归档
- **自动清理**：可选保留或删除原始文件
- **自动窗口**：可选自动关闭 Bandizip 弹窗

#### 配置方式

- `CONFIG` 字典集中管理全部参数
- 支持 `.txt → 文档` 等自定义扩展名→目录名映射

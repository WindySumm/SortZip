# SortZip

> 文件分类 → 可选命名规则 → 分组加密压缩，一站式批处理工具。

基于 **Bandizip** 命令行实现自动分卷、密码保护和二次打包归档。

---

## 快速开始

**环境要求：** Windows + [Bandizip](https://www.bandizip.com/)（加入 PATH）

直接运行 `dist/SortZip.exe` 即可启动图形界面。

---

## 功能流程

1. **分类** — 按扩展名将源文件归类至子目录（支持自定义分类名）
2. **命名规则**（可选）— 每个文件夹可独立配置模板（`{n}`、`{ext}`、`{folder}`、`{original}`），支持预设快速填充
3. **对照表**（可选）— 勾选后在各文件夹生成 `List.txt`，记录序号、原文件名、新文件名（及压缩包名），`\t` 分隔可直接导入 Excel
4. **分组压缩**（可选）— 每 N 个文件一组，Bandizip 带密码压缩，自动或手动分卷；可仅执行一次压缩（无 `-First` 后缀）或两次压缩（先分卷后归档）
5. **二次压缩**（可选）— 分卷文件再次打包为单文件归档（默认后缀 `.zipp`，可自定义）

执行前自动校验：
- 目标文件夹冲突检测
- 源文件类型匹配检测
- 命名模板冲突检测

## 界面导航

左侧导航栏切换，共 6 个页面：

| 页面 | 功能 |
|------|------|
| **文件** | 文件选择（源/目标文件夹、包含子文件夹）、密码设置（压缩密码/确认密码/记住密码）、分卷设置（每包文件数/分卷大小/启用分卷） |
| **映射** | 扩展名→文件夹名映射，分组勾选（视频/音乐/图片/文档/程序/字体/压缩包） |
| **命名** | 排序设置（7 种排序预设）、每个映射文件夹独立命名规则（支持预设模板）、压缩后重命名（替换后缀） |
| **预览** | 标签页切换文件夹，显示所有文件重命名前后对比 |
| **设置** | 深色模式、使用手册、版本信息 |
| **开始** | 压缩设置（一次压缩、二次压缩、压缩格式）、其他选项（保留原始文件、输出目录 List.txt、自动关闭 Bandizip）、进度条、执行/取消、实时日志、统计报告 |

## 注意事项

- **仅支持 Bandizip**，必须安装并加入 PATH
- 密码非空时 ≥ 8 位
- 命名规则默认禁用，需手动启用
- 模板留空 = 不重命名，保留原始文件名
- 压缩包名始终为组序号格式（`1-4`、`5-8`），不受文件命名规则影响
- 排序方式位于「命名」页面顶部，可选 7 种排序预设
- 取消勾选「启用分卷」可强制单文件输出，不进行分卷
- 勾选「输出目录」后各文件夹生成 `List.txt`，`\t` 分隔，可直接拖入 Excel；未启用压缩时自动省略「所属压缩包名」列
- 勾选「保留原始文件」时源文件仅复制，压缩后副本自动清理
- 取消勾选「一次压缩」可跳过所有压缩步骤，仅执行分类与重命名
- 首次使用建议先小范围测试

# SortZip

> 文件分类 → 顺序重命名 → 分组加密压缩，一站式批处理工具。

仅支持结合 **Bandizip** 使用，基于其命令行实现自动分卷、密码保护和二次打包归档。

---

## 功能流程

1. **分类** —— 按扩展名将源文件夹中的文件自动归类至子目录（支持自定义分类名）
2. **重命名** —— 每个目录内按文件名或修改时间排序，顺序重命名为 `1.ext`、`2.ext`…
3. **分组压缩** —— 每 N 个文件为一组，使用 Bandizip 带密码压缩：
   - 自动检测分卷大小（<4GB 时分半，≥4GB 时每卷 4GB）
   - 支持手动指定分卷大小
4. **二次打包**（可选）—— 将分卷文件再次压缩为一个 `.zipp` 文件

---

## 快速开始

### 环境要求

- Windows 操作系统
- [Bandizip](https://www.bandizip.com/) 已安装并加入 PATH（或配置完整路径）

### 使用方法

直接运行 `dist/SortZip.exe` 即可启动图形界面。

## 配置说明

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `src` | `str` | — | 源文件夹路径（首次为空，自动记住上次路径） |
| `dest` | `str` | — | 目标根目录（首次为空，自动记住上次路径） |
| `group_size` | `int` | `1` | 每组文件数 |
| `password` | `str` | — | 压缩密码（留空无密码，非空时需 ≥ 8 位） |
| `volume` | `str\|None` | `None` | 手动分卷大小如 `100m`，`None` 为自动 |
| `custom_names` | `dict` | 视频/音乐/文档/图片等 | 扩展名 → 文件夹名映射，UI 默认预置 7 条 |
| `sort_by` | `str` | `name` | 排序方式：`name` 或 `mtime` |
| `keep_files` | `bool` | `False` | 压缩后是否保留原始文件 |
| `double_compress` | `bool` | `True` | 是否二次打包为 `.zipp` |
| `auto_close` | `bool` | `True` | 是否自动关闭 Bandizip 窗口 |

## 注意事项

- **本工具仅支持 Bandizip**，请确保已安装并加入 PATH
- 密码非空时必须 ≥ 8 位，否则运行时会提示
- 首次运行建议先在小范围测试
- `keep_files = False` 时会删除原始文件，请谨慎配置
- `.zipp` 是脚本自用的二次打包后缀，非标准格式
